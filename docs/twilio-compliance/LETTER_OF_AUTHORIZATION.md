### SMS Program Description

**Program Name:** TalkCRM Phone Verification

**Program Type:** Authentication / One-Time Password (OTP)

**Message Purpose:**
TalkCRM uses SMS exclusively for user authentication and phone number verification. When users sign up for TalkCRM or add a new phone number to their account, they receive a one-time verification code via SMS to confirm ownership of the phone number.

**Sample Message Content:**
```
Your TalkCRM verification code is: 123456. This code expires in 10 minutes.
```

**Message Frequency:**
- One (1) SMS per verification attempt
- Users typically receive 1-2 verification messages during initial account setup
- Occasional verification messages when adding additional phone numbers

**Opt-in Method:**
Users explicitly initiate the verification process by entering their phone number in the TalkCRM signup form and clicking "Send Verification Code."

**Opt-out Method:**
- Users can reply STOP to any message to opt out
- Users can remove their phone number from their TalkCRM account settings
- Contact support@talkcrm.ai
